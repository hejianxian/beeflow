# beeflow
A front-end build tool with webpack.

### beeflow dev

需要通过cli获取到pwd来设置以下值

* publicPath
* contetnBase
* express staticPath

相关工具:

* http-proxy-middleware
* connect-history-api-fallback
* webpack-hot-middleware
* webpack-dev-middleware
* html-webpack-plugin-after-emit

需要暴露到beeflow配置的有:

* $beeflow external
* server proxy
